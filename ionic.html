Application Ionic

Présentation :

Cette application permet d'avoir la listes des pharmacies, supprimer, ajouter ou supprimer une pharmacie.
Cette application permet d'avoir le compte rendu des medecins.
Diverses informations sont également disponible comme la météo en temps réel d'une ville souhaitée grâce à une API Rest disponible gratuitement via le site OpenWeatherMap.
Un onglet "Ville" permettant d'avoir des photos de la ville en question et un onglet "Pays" permettant d'avoir tout les drapeaux des pays du monde via une API.
